# 🚴 VELOS PARIS DATA - UI DASHBOARD (V2)

**Projet :** Visualisation des données cyclables parisiennes (Stations, Flux, Pistes)
**Engine :** Propulsé par `RenovateApp_Launcher_2/engine` (TAYIER OS)

## 🎯 OBJECTIFS
Cette interface React (Vite) doit consommer l'API du Engine pour afficher :
1.  Carte en temps réel des vélos disponibles (Leaflet).
2.  Statistiques d'usage par arrondissement (Recharts).
3.  Prédictions IA du TAYIER OS sur l'encombrement.

## 🚀 DÉMARRAGE RAPIDE
Depuis la racine du projet, lancez simplement :
```bash
python DEMARRER_2.py
```
Cela lancera le Moteur Python ET cette Interface React (Port 5174).

## 🛠️ INSTALLATION MANUELLE
```bash
cd RenovateApp_Launcher_2/ui2/react-app
npm install
npm run dev
```
